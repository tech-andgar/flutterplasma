# Flutter Plasma

Flutter app demonstrating Flutter web rendering.

## Routes

- `/`: Default demo
- `/nocredits`: Demo stops before credits
- `/showroom`: Enters Show Room
- `/showroom/:index`: Jump to specific Show Room example
